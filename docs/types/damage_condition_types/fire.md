# Fire

[Damage Condition Type](../damage_condition_types.md)

Checks whether the damage is fire damage (this includes lava damage!).

Type ID: `apoli:fire`


### Fields

_None._


### Examples

```json
"damage_condition": {
    "type": "apoli:fire"
}
```
