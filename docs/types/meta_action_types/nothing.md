# Nothing

[Meta Action Type](../meta_action_types.md)

Does nothing. Can be used as a backup in case an action is not optional in some place.

Type ID: `apoli:nothing`

### Fields

_None._

### Examples

```json
"entity_action": {
    "type": "apoli:nothing"
}
```

This example does.. nothing.
<br>

```json
"bientity_action": {
    "type": "apoli:nothing"
}
```

This example also does.. nothing.
