# Crafting Table

[Entity Action Type](../entity_action_types.md)

Brings up the Crafting Table GUI as if the player has interacted with one.

Type ID: `apoli:crafting_table`


### Fields

_None._


### Examples

```json
"entity_action": {
    "type": "apoli:crafting_table"
}
```
