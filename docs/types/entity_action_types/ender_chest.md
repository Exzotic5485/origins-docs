# Ender Chest

[Entity Action Type](../entity_action_types.md)

Brings up the Ender Chest GUI as if the player has interacted with one.

Type ID: `origins:ender_chest`


### Fields

_None._


### Examples

```json
"entity_action": {
    "type": "origins:ender_chest"
}
```
